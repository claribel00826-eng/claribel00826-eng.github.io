# 标注文档 05：意图 · 槽位路由（1.2.0 版）

> **填充数据**：只写**表单里的字段名**（与文档 04 各弹窗/抽屉一致）；**产品**与**产品规格**分列，不写功能、单号、卡片名等。  
> 该步**没有可填表单**时填「—」。  
> **落页**：输入区 · 面板「意图」。流程内其余槽位见文档 04。

---

## 一、处理顺序

① 识别功能意图 → ② 当前企业（顶栏）→ ③ 当前客户（若要）→ ④ 意图槽齐 → 跳转并**预填对应表单字段**

---

## 二、功能意图识别

| 功能意图 | 识别方式 | 要客户 |
|----------|----------|--------|
| 切换客户 | 切换/换/选择/选客户 | 否 |
| 今日待跟进 | 待跟进、今天要跟… | 否 |
| 写跟进 | 写跟进、给×写跟进 | 是 |
| 方案速配 | 配方案、选品；或一句话品名/数量 | 是 |
| 产品报价 | 报价、去报价、按方案报价（分步）；含单价/折扣的长句（可一句话出卡） | 是 |
| 确认下单 | 下单、下+品名、按报价单… | 是 |
| 未识别 | 以上都不命中 | — |

### 3.5 产品报价

| 已读到 | 跳转 | 填充数据 |
|--------|------|----------|
| 功能意图=产品报价 | 选客户引导卡 | — |
| + 客户（口令「去报价」「报价」「按方案报价」） | 1 个方案→逐项报价；多个方案→选择方案卡；无方案→报价来源卡 | 客户；选定方案后预填产品、规格、数量 |
| + 按方案/直选一句话（含单价或折扣） | 报价单卡 | 产品、产品规格、产品数量、本单报价、报价单模板 |

> 意图层单价字段名为 **本单报价**（与逐项报价弹窗表单项一致）；**无折扣**字段。
