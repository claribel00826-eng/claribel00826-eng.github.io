# 标注文档 05：意图 · 槽位路由（v1.2.0）

> **范围**：对话区输入的意图识别、分步槽位、缺槽引导、跨功能、话术选客；与文档 01 待跟进/写跟进、文档 02 主链路一并验收。  
> **填充数据**：面板表格「填充数据」列只写**表单字段名**；无表单填「—」。完整字段与缺槽话术见 **文档 04**。  
> **落页**：输入区「意图」钉 → 意图分步槽位面板（`chat-llm`）。

---

## 一、处理顺序

① 识别功能意图 → ② 当前企业（顶栏，登录带出）→ ③ 当前客户（方案/报价/下单及写跟进等须）→ ④ 定当前步骤与界面焦点 → ⑤ 跳转并**预填对应表单字段**

**解析范围优先**（与《用户输入与意图填槽需求》§3.2 一致）：抽屉打开 > 需求提示卡 > 入口选择卡 > 列表点选 > 来源卡 > 流程内卡片 > 入口口令。

## 四、下一步五类（验收摘要）

| 类型 | 何时 |
|------|------|
| **推进** | 自当前步起无更早缺槽；方案/报价单检索唯一 |
| **消歧** | 方案或报价单命中多条 → 选择列表 |
| **缺槽引导** | 【待填写】+ 打开首缺槽对应界面 |
| **跨功能** | 切换能力且可沿用数据 → 弹窗 |
| **能力说明** | 无法识别或按钮已失效 |

缺槽引导话术与「定位界面」列以意图钉面板 **各功能「2. 缺槽引导」** 为准，与文档 04 同句。

## 五、话术选客（客户槽）

| 结果 | 处置 |
|------|------|
| 唯一命中 | 直接切换客户，不弹抽屉 |
| 多家命中 | 打开选客户抽屉，搜索预填关键字 |
| 未命中 | 提示核对名称或点「选择客户」 |

列表可见范围见文档 02 模块 2.1。
