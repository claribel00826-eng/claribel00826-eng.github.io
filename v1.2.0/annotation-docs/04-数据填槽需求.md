# 标注文档 04：数据填槽需求（v1.2.0）

> **范围**：各弹窗/抽屉/卡片上的**表单字段**及**缺失时引导话术**；意图层缺槽时引用本表。  
> **意图分步、跳转、话术选客**：见 **文档 05**；界面操作见 **文档 02**。  
> **原型**：演示校验 + 意图引导；正式环境以字段必填规则为准。  
> **落页**：意图钉缺槽表与各模块标注「查询/校验」引用本表原句。

---

## 一、使用说明

1. 无「场景描述」列：用户可随时切换客户或功能，槽位按**功能模块 + 所在界面**列出。  
2. **是否必填**：「条件必填」表示仅在某分支下必填（如按方案报价须已有方案）。  
3. 与文档 05 分工：本文 = **字段名 + 缺槽话术**；文档 05 = **已读到 → 跳转 → 预填哪些字段**。
