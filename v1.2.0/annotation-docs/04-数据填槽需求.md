# 数据填槽需求 · 销售助手（1.2.0 版）

> 覆盖本版主链路（方案速配、产品报价、确认下单）及对话内仍可用的关联能力。  
> **意图层**（分步槽位读取、合并会话态、缺槽引导、卡片语音填槽、无匹配兜底）见 **文档 05**。本文为各弹窗/抽屉**表单字段**清单及**缺失时引导话术**（意图层缺槽时引用本表）。
