整体：主链路

左栏：入口触点（不按流程连线）。右栏：仅画 大模块之间的串联（不展开待跟进内详情、方案/报价内历史 PDF 等子步骤；子能力见各 F 章节）。串联为：首页 / 服务号模板消息 → 待跟进列表 → 方案速配（选品、购物车、保存方案）→ 产品报价 → 交期评审 → 按结论 生成订单、调整方案、插单申请；生成订单 后可再使用 订单复制、订单变更、订单查询、客户服务（与宫格并联，虚线表示非必经）。报价为必经语义环节；交期可跳过须经确认（见报价 PDF）。与 PRD §4、§6 一致。

入口：首页 / 消息模板 / 待跟进

首页宫格不出现「客户开拓」；开拓与同一客户池亦可通过「待跟进客户」进入列表后再进详情。

服务号：每日固定 1 条摘要「今日待跟进 n 个，点击查看」→ /development（可不携带 customer_id）。另支持 URL 带 customer_id 且已登录：选中该客户 后进入 /development。

每日摘要 → /development · customer_id 深链 → 客户开拓

首页宫格（当前实现；权限可裁剪）：

方案速配 产品报价 交期评审 生成订单 订单复制 订单变更 订单进度 客户服务

首页另有底部固定语音/文字输入：单次意图识别，槽位客户（可选）+ 功能，无多轮澄清；详见 F03与 F15。

语音上传转写 文字直达 意图浮层

F03 首页与待跟进

聚合功能宫格（无「客户开拓」项）、待跟进条与最近客户。底部固定语音/文字输入（与 F15）：提交后经单次意图识别输出结构化槽位——客户（非必填）+功能，以意图浮层展示并可关闭；可在规则允许时自动执行切客/跳转。不做多轮澄清追问。最近客户：卡片副文展示联系人 + 最近访问业务页（由恢复点路径映射），不展示新老客户等分类标签；记录最近一次业务 URL，点击恢复；无记录默认方案速配选品。（当前打点可为 localStorage，上线可对齐服务端。）

F04 客户开拓

支持系统推送待跟进提醒与销售主动进入（与首页待跟进列表同一口径）。列表中选客户后进入客户详情（客户编码、名称、性质、所属类别、结算客户、等级及跟进信息列表）；在详情中可进入写跟进或方案速配。

F05 方案速配

单页内 选品 Tab / 购物车 / 保存方案；顶栏右侧进方案历史。保存后 方案 PDF 预览导出。历史版本可变更回填购物车。新客户与老客户在选品步的业务口径不同（见下图），购物车及之后一致。方案速配、方案 PDF、方案历史三路由不展示底部全局语音/文字输入条（见 F15）。

F08 产品报价

产品报价顶栏含历史报价。列出当前客户全部方案选一 → 询价（最低/最新售价参照 + 用户填本次单价）→ 生成报价单 → 报价 PDF（预览/导出，技术同方案 PDF）。历史报价列表可预览/导出（进入报价 PDF）或重新报价（回到询价并带入关联方案）。PDF 底栏：交期评审、直接下单（风险确认）；历史列表仅从报价页顶栏进入。

F09 交期评审

选择客户（当前工作客户）→ 选择报价单（该客户下全部报价）→ 输入期望交期 → 生成预测：引擎给出是否预计按期交付、是否齐套、延期卡点；话术摘要可由大模型归纳展示。评审通过后方可进入生成订单；否则可走调整方案或插单。

F10 生成订单

一报价可生成多笔订单。若在报价侧已确认跳过交期，须留有弹窗确认记录。提交订单前仍校验交期评审状态；未完成时须风险提示并再次确认。

F11 订单复制

将历史订单行项目复制为草稿并写入当前客户购物车，用于快速改单与复购。复制完成后进入方案速配的购物车或方案子步继续编辑。

F15 全局语音与意图
