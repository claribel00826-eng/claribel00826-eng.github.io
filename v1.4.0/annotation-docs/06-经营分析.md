## 二、产能分析 · `card-capacity`

### 2.1 内容

1. **标题**：「产能分析」
2. **块 A · 负荷描述**（销售助手计算 + 展示，只读）
3. **Tab 栏**：「甘特图」|「订单详情」（详情 Tab 可带订单号后 6 位徽章）
4. **块 B · 甘特 Tab**（默认）：
   - 一行一产线；首屏 1 天，横滑查看更多
   - 时间轴粒度：**小时**（无粒度选择控件）
   - 非工作灰带、当前时刻线、占用块色块/锁/预
   - helper：「点击占用块查看订单详情；点「甘特图」继续浏览」
5. **块 C · 订单详情 Tab**（见 §2.2）

### 2.2 查询逻辑

- 前置：顶栏已选当前客户
- 甘特 + 占用详情：**已有后台能力直返**（功能描述 §1.1.9）
- 负荷描述：销售助手 **基于甘特数据计算** 排产至日期、平均负荷率（今日～排产至日期，工作实际小时数÷可工作小时数；功能描述 §1.1.10）
- 演示阶段读 **本地聚合演示数据**（非真实接口）

### 2.3 交互逻辑

- 点占用块 → 填详情 → **自动切到「订单详情」Tab**
- 点「甘特图」Tab → 回到甘特，**横滑位置保留**；刚查看块闪烁高亮约 2s
- 未选块就点「订单详情」Tab → **轻提示**
- 禁止拖拽改排

## 三、订单详情 Tab · `card-capacity-block-detail`

### 3.1 内容

- 摘要行：「当前查看：{订单号} · {产线}」
- 16 字段标签-值列表（只读，可滚动）

### 3.2 交互逻辑

- 由甘特 Tab 点块进入；也可在已选块后手动切 Tab 查看
- 切回「甘特图」Tab 继续浏览
