## 用户测试引导

1. 打开 [v1.5.0/index.html](./index.html)，登录 `demo` / `123456`
2. **最近访问**：点「库存查询」→ 初始化页面 → 首屏最近访问应出现「库存查询」
3. **新增客户**：顶栏「切换」→「+ 新增客户」→ 填表提交 → 顶栏显示新客户
4. **缺客户引导**：未选客户时点「产品报价」→「新建客户」
5. **话术**：输入「新增客户」打开表单
6. **自由项自定义**：
   - 做方案 → 选品卡勾选含颜色/大小的产品 → 颜色下拉选「其他…」→ 下拉变为行内输入框 → 输入「玫瑰金」→ 确定
   - 下拉应显示「玫瑰金（自定义）」；同产品另一行颜色下拉也应出现该选项
   - 点「预览方案」→ 方案预览卡可继续改自由项并添加自定义项
   - 确认下单 → 选品 → 订单购物车同样支持「其他…」
