## 三、三块摘要

| 中文模块名 | 一、展示与口径 | 二、范围 | 三、交互 |
|------------|----------------|----------|----------|
| 产能分析 | 负荷描述；甘特 Tab（产线行、占用块）；详情 Tab 见子卡 | 全厂产线；不要求选客户 | 点块切 Tab；禁拖拽改排 |
| 产能分析 · 订单详情 Tab | 当前查看摘要；订单字段列表 | 随占用块一并返回 | 甘特点块切入；回甘特保留横滑 |
| 业务分析 | 摘要行；双 Tab；段选；Top10 排行；结论子卡 | 全部客户企业；本年 1/1～今日 | 切 Tab/指标；无下钻 |
| 业务分析 · 一句话结论 | Tab 下一句经营判断 | 仅总金额口径；全量排行算占比 | 随 Tab 换句；段选不重算 |
| 库存查询 | 加粗合计行；两行规格列表 | 全仓合计；不按客户过滤 | 欢迎区/技能/话术进入；只读 |
| 回款分析 | 加粗户数摘要；2×2 指标块 | 全部客户汇总 | 欢迎区/技能/话术进入；只读 |

详细指标与字段见功能需求说明 **§1.1**；`annotation-spec-data.js` 与验收面板同源。
