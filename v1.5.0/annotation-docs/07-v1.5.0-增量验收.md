## 三、三块摘要表

| 中文模块名 | 一、展示与口径 | 二、范围 | 三、交互 |
|------------|----------------|----------|----------|
| 首屏 · 最近访问 | 功能名 + 相对时间；次行进度摘要；最多 5 条 | 不需客户的功能可独立展示 | 点击续接功能 |
| 对话流 · 首屏三块 | 欢迎/待跟进/最近访问同条消息 | — | 下一张助手页后整条首屏失效 |
| 选客户卡 · 本版变更 | 可见范围 + 我负责且无订单；底部新增客户 | 当前企业客户子集 | 无结果提示；打开新增表单 |
| 新增客户全链路 | 表单字段、地区选择器、多入口 | 性质固定为客户 | 提交设当前客户；取消/提交保留快照 |
| 选品/预览/购物车/逐项报价 | 一行一自由项；「其他…」会话自定义 | 同产品同维度共享 | 行内输入替换下拉 → 确定/取消 |

## 四、验收抽验建议

1. 点「库存查询」→ 初始化 → 首屏出现「最近访问」。  
2. 技能条「+ 新增客户」→ 提交 → 选客户列表可见新客户。  
3. 方案选品 → 颜色「其他…」→ 自定义取值 → 同产品其他行可见。  
4. 开启「设计标注」→ **仅**上表模块出现「标注」钉；其余历史卡片无钉。
