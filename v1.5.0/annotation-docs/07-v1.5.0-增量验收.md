## 三、三块摘要表

| 中文模块名 | 一、展示与口径 | 二、范围 | 三、交互 |
|------------|----------------|----------|----------|
| 首屏 · 最近访问 | 功能名 + 相对时间；次行进度摘要（有客户时前缀客户名）；按功能去重最多 5 条 | 不需客户的功能可独立展示 | 点击续接功能；有客户记录且仍可见时恢复客户 |
| 选客户卡 · 新增客户 | 选客户 inline 卡底部按钮 | — | 打开新增客户表单卡 |
| 技能条 · 新增客户 | 底栏 chip，样式同其他技能 | — | 模拟发送「新增客户」并打开表单卡 |
| 缺客户 · 引导卡 | 需客户技能未选客户时展示 | — | 选择客户 / 新建客户 |
| 客户 · 新增表单卡 | 编码只读；名称必填；类别一级必选；地区内嵌选择器 | 性质固定为客户 | 提交成功设当前客户；取消/提交保留只读快照 |
| 新增客户 · 地区选择器 | 搜索优先；可浏览树；须确认才写入 | 内嵌表单，不单独推卡 | 展开/收起；确认回填 |
| 选客户卡 | 搜索 + 列表；可见：老客户 ∪ 公海新客户 ∪ 我负责且无订单 | 当前企业客户子集 | 点选更新顶栏；无结果提示；底部新增客户 |
| 方案选品卡（自由项） | 每产品独立卡片；颜色、大小各占一行；末项「其他…」可添加本会话自定义取值 | 同产品同维度会话共享 | 「其他…」→ 行内输入替换下拉 → 确定/取消 |
| 方案预览卡（自由项） | 已勾选品名 + 自由项 + 数量纵向排列 | 改动写入方案上下文 | 同选品卡自定义自由项 |
| 订单购物车（自由项） | 已选品 + 自由项 + 数量 | — | 同选品卡；下一步逐项报价 |

## 四、验收抽验建议

1. 登录后点「库存查询」→ 初始化再进 → 首屏「最近访问」出现该功能。  
2. 技能条「+ 新增客户」→ 填表提交 → 顶栏显示新客户；选客户列表可见该客户。  
3. 做方案选品 → 颜色选「其他…」→ 输入自定义取值 → 确定 → 同产品其他行下拉可见。  
4. 预览方案 / 订单购物车可继续添加自定义自由项。  
5. 开启「设计标注」→ 上述模块右上角应出现「标注」钉，面板文案与上表一致。
