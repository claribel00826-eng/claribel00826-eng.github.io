# 竞品调研结论（跨版本）

> 来源：阶段 R，已写入各版本 PRD 背景章节。  
> 产品：**制造业外勤销售 · 纯对话智能体 H5**

## 竞品对比

| 竞品 | 可借鉴 | 不采纳 |
|------|--------|--------|
| 豆包 | 底部语音/文字输入；输入框上方横滑 Skill/能力条；说一句即响应 | C 端过大留白 |
| YonClaw | 企业上下文 + 业务结构化卡片；B 端信息密度 | — |
| 纷享销客 | 待跟进字段、写跟进表单、外勤习惯 | 多 Tab/多页跳转 |

## 选定方向

**方案 A**：豆包式轻壳 + B 端业务卡片，视觉参考 YonClaw。

## 体验红线（全局）

- 面向销售人员的界面符合大众 H5 习惯，不出现「标注模式」等产品化入口。
- 设计标注仅用于交付验收：默认隐藏，可通过 `?spec=1` 或快捷键打开（不对最终用户宣传）。
