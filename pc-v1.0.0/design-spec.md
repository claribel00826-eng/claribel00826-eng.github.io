# PC 端设计规范 · v1.0.0

> 意图 QA 对维护模块。H5 销售助手演示仍见 `v1.5.0/design-spec.md`。

## 布局

| 项 | 值 |
|----|-----|
| 最小宽度 | 1280px |
| 页面背景 | `#F5F5F5` |
| 内容 padding | 24px |
| 侧栏宽度 | 240px |
| 顶栏高度 | 56px |

## 色彩

| Token | 值 | 用途 |
|-------|-----|------|
| Primary | `#108482` | 主按钮、选中、链接 |
| Primary Hover | `#0E7371` | |
| Surface | `#FFFFFF` | 卡片、表格、抽屉 |
| Border | `#E4E4E7` | 表格线、输入框 |
| Text | `#18181B` | 正文 |
| Text Muted | `#71717A` | 辅助说明 |
| Warning | `#D97706` | multi-Q 歧义标记 |
| Danger | `#DC2626` | 删除、校验失败 |

##  typography

| 级别 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 20px | 600 |
| 表格头 | 14px | 600 |
| 表格体 | 14px | 400 |
| 辅助 | 12px | 400 |

## 组件

| 组件 | 规格 |
|------|------|
| 卡片 | 圆角 8px，padding 20px，无重阴影 |
| 表格行 | 高 48px |
| 主按钮 | 高 36px，圆角 8px |
| 输入框 | 高 36px，圆角 8px |
| 编辑抽屉 | 宽 480px，右侧滑出 |
| 功能标签 | 按 A 枚举配色，浅底 + 深字 |
