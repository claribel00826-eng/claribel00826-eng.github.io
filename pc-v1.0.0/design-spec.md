# PC 端设计规范 · v1.0.0

> 意图 QA 对维护模块。H5 销售助手演示仍见 `v1.5.0/design-spec.md`。

## 布局（v1.0.0 改版）

采用常规 B 端后台结构：**顶栏导航 + 左侧模块栏 + 主内容区**。

| 区域 | 说明 |
|------|------|
| 顶栏 `app-nav` | 48px，品牌 |
| 侧栏 `app-sidenav` | 200px，仅模块导航 |
| 页面头 `page-header` | 标题/说明 + **试跑验证、导入 Excel、导出 Excel、新增** |
| 筛选条 `filter-bar` | 横向：搜索 Q、主功能、状态、歧义 Q、重置 |
| 数据概览 `stats-bar` | 列表上方横向三指标：QA 对 / 唯一 Q / 歧义 Q |
| 内容卡 `content-card` | Tab + 表格 + 分页（每页 10 条） |
| 试跑弹窗 `modal-trial` | 页头「试跑验证」或列表行内「试跑」打开 |
| 导入弹窗 `modal-import` | 页头「导入 Excel」：模板下载 + 拖拽/选择上传 |
| 编辑抽屉 | 480px 右侧滑出（不变） |

表格列顺序：勾选 → **序号** → Q → A → 备注 → 状态 → 操作。序号 / Q / A 列头常显 ↑↓，默认按序号升序。

| 项 | 值 |
|----|-----|
| 最小宽度 | 1280px |
| 页面背景 | `#F5F5F5` |
| 主内容 padding | 20px 24px |
